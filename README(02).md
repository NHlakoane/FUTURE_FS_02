# FUTURE_FS_02 – CRM System

## 📌 Project Overview

This project is a full-stack Customer Relationship Management (CRM) system developed as part of the Future Interns Full Stack Web Development track.

The system allows businesses to manage customer leads, track interactions, and monitor conversion progress efficiently.

---

## 🚀 Features

* User authentication (Login/Register)
* Add, update, and delete leads
* Lead status tracking (New, Contacted, Converted)
* Notes management for each lead
* Dashboard with basic analytics
* RESTful API integration

---

## 🛠️ Tech Stack

**Frontend:**

* React (Vite)
* Axios
* CSS

**Backend:**

* Node.js
* Express.js
* MongoDB

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/NHlakoane/FUTURE_FS_02.git
cd FUTURE_FS_02
```

### 2. Backend setup

```
cd backend
npm install
npm start
```

### 3. Frontend setup

```
cd frontend/vite-project
npm install
npm run dev
```

---

## 🔐 Environment Variables

Create a `.env` file in the backend folder and add:

```
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```
